# 学院教师评价系统

#### 介绍
根据《教师教学质量评价实施方案（试行）》开发教师评价系统。

#### 软件架构
使用人人开源社区项目renren_security为基础开发。


#### 安装教程

1. 修改数据库连接地址、配置文件内的端口、ContextPath，打包te-admin.jar。
2. 使用setsid java -jar te-admin.jar，后台运行项目。

#### 使用说明

1. 环境要求 JDK1.8、Mysql8.0
